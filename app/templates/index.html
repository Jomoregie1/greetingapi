Available Greeting Categories

Our API offers a wide range of greeting categories to suit various occasions and relationships. Below is the list of available categories:

  • Birthday_Boyfriend: Birthday greetings specifically tailored for boyfriends.
  • Birthday_Girlfriend: Special birthday messages for girlfriends.
  • Birthday_Love: Romantic birthday greetings for a significant other.
  • Birthday_Wife: Heartfelt birthday wishes for wives.
  • Birthday_Bestfriend: Fun and thoughtful birthday messages for best friends.
  • Birthday_Brother: Birthday greetings for brothers, from heartfelt to humorous.
  • Birthday_Dad: Warm and appreciative birthday wishes for fathers.
  • Birthday_Mom: Loving birthday messages for mothers.
  • Birthday_Sister: Birthday wishes for sisters, celebrating the special bond.
  • Christmas_Boyfriend: Festive Christmas messages for boyfriends.
  • Christmas_Girlfriend: Christmas greetings full of love and cheer for girlfriends.
  • Christmas_General: General Christmas greetings suitable for anyone.
  • Morning_Romantic: Romantic messages to start the day with love.

Understanding API Query Parameters

limit
Specifies the number of results to return. Default is 10.
offset
Indicates the starting point for the query, used for pagination.
query
Used for search queries to filter greetings based on specific keywords.

GET a list of greetings

GET https://seasonsgreetingsapi/v1/greetings/?category={category}&limit={limit}&offset={offset}

                        
                        
                            curl https://seasonsgreetingsapi/v1/greetings/?category={category}&limit={limit}&offset={offset}

Python

                        
                        
                            import requests

                            category = "YOUR_CATEGORY" # Replace with your chosen category
                            limit = 10 # Adjust the limit as needed
                            offset = 0 # Adjust the offset as needed

                            url = f"https://seasonsgreetingsapi/v1/greetings/?category={category}&limit={limit}&offset={offset}"
                            response = requests.get(url)
                            json_data = response.json()
                            print(json_data)
